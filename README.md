## Глава 3 Введение в программирование 

Ветвь - строительные блоки составных конструкций. 
Ветвь состоит из двух или более строк кода: 
заголовка, за которым следует тело

Выражение: код с оператором между двумя операндами 

Заголовок - строка кода в ветви, содержащая ключевое 
слово, за которым следует двоеточие, и последовательность 
из одной или нескольких строк с отступом. 

Инструкция - команда или вычисление 

Объект - значение с тремя свойствами - идентификатором, типом данных и значением 

Операнд - значение с любой стороны оператора

Оператор - символы, использующее в выражении с операндами 

Простая инструкция - инструкция на одной строке кода 

Составная инструкция - инструкция, занимающая более одной строки кода обычно

Тело - строка кода внутри ветви
